# Проект hw05_final

### Описание проекта

#### Сервис для публикаций — это веб-приложение на Django, которое позволяет пользователям создавать публикации, регистрироваться, комментировать посты и подписываться на авторов. Моя основная задача в этом проекте заключалась в тщательном тестировании функционала приложения.

### Тестирование

#### Покрыты тестами:
#### Шаблоны. Проверяется, что url-адреса используют соответствующие шаблоны и все шаблоны сформированы с правильным контекстом.
#### Пагинатор. Для каждого указанного URL-адреса проверяется, что на каждой странице количество постов соответствует ожидаемому значению.
#### Урлы. Проверяется, что URL-адреса возвращают правильные коды состояния HTTP.
#### Модели. Проверяется, что метод __str__ возвращает ожидаемые значения (метод __str__ в Django отвечает за строковое представление объекта модели).
#### Формы. Проверяется функциональность создания, редактирования постов и добавления комментариев.
#### Подписки. Проверяется функциональность подписок и ленты новостей. Корректность подписки/отписки пользователей, отображение и не отображение постов пользователей, на которых текущий пользователь подписан или не подписан.
#### Кэш. Проверяется, что кэширование корректно сохраняет и обновляет данные.

### Инструкция по запуску на локальной машине

#### 1. Клонируем репозиторий: git clone git@github.com:Eduard-Golovash/hw05_final.git
#### 2. Переходим в директорию hw05_final: cd hw05_final
#### 3. Создаем и активируем виртуальное окружение:
#### - python -m venv venv
#### - source venv/Scripts/activate
#### 4. Устанавливаем зависимости: pip install -r requirements.txt
#### 5. Делаем миграции: python manage.py migrate
#### 6. Создаем суперпользователя: python manage.py createsuperuser
#### 7. Запускаем сервер: python manage.py runserver
